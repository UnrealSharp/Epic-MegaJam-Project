Slime Guzzler is a wave based arena shooter, where you fight your way through waves of enemy slimes, and upgrade your player as you go.  Should... When, you succumb to overwhelming numbers, well, you start again from wave number one, here we go again!  

The game was made in one week for the 2025 Epic MegaJam using **UnrealSharp**.
